# Midnight Galaxy

深いパープルと神秘的なトーンが特徴の、ドラマチックでコズミックなテーマ。インパクトのあるプレゼンテーションに最適です。

- **Deep Purple**: `#2b1e3e` - 深みのあるダークベース
- **Cosmic Blue**: `#4a4e8f` - 神秘的なミッドトーン
- **Lavender**: `#a490c2` - ソフトなアクセントカラー
- **Silver**: `#e6e6fa` - ライトハイライトとテキスト
